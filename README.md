# HandwrittenMathVerifier

An AI-powered Handwritten Mathematical Expression Recognition and Verification system.

## Features

- Handwritten mathematical expression recognition
- OCR using deep learning
- CNN/CRNN-based feature extraction
- Mathematical expression verification
- Equation parsing
- Result visualization

## Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- SymPy
- Matplotlib

## Project Structure

```
HandwrittenMathVerifier/
│
├── dataset/
├── models/
├── train.py
├── predict.py
├── requirements.txt
├── README.md
└── .gitignore
```

## Installation

```bash
git clone https://github.com/SubhaOp/HandwrittenMathVerifier.git

cd HandwrittenMathVerifier

pip install -r requirements.txt
```
